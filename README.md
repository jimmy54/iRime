# iRime輸入法         注：禁止代码原封不动打包上传到AppStore

=====
基於著名的[Rime]輸入法框架，
旨在保護漢語各地方言，
音碼形碼通用輸入法平臺。



- [点击到AppStore下载](https://itunes.apple.com/cn/app/irime输入法/id1142623977?mt=8)
